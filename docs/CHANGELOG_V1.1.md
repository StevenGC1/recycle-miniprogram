## 五、交互功能新增

### 1. 首页地址选择 —— 真实地图选点
- `pages/index/index.js` 的 `onChooseAddress` 改为调用 `wx.chooseLocation`
- 如果模拟器/环境不支持定位权限导致调用失败，会弹窗提供"使用模拟地址"作为兜底，保证演示流程不会卡住
- **重要**：`wx.chooseLocation` 需要小程序在微信公众平台后台开通"位置信息"接口权限，且部分机型/版本首次使用会弹出授权确认框。如果你在真机上测试时被拒绝了定位授权，需要去手机「微信 - 设置 - 隐私 - 小程序授权管理」里重新打开"位置信息"权限。

### 2. 新增"亲友代下"完整页面
- 新增 `pages/familyBind/familyBind`：填写称呼 + 地图选点地址 + 选填电话 → 保存绑定（存在页面内存里，不持久化）
- 已绑定列表里每条都有【为他预约】按钮（原型阶段先做提示弹窗，后续可以做成"跳转首页并预填该地址"）
- `pages/mine/mine.js` 的 `onAddFamily` 从原来的"提示弹窗"改为 `wx.navigateTo` 真实跳转到这个新页面
